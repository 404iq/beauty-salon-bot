# 💅 Бот для записи в бьюти-салон

Telegram-бот для онлайн-записи клиентов в салон красоты. Написан на Python + aiogram 3.x.

## Функционал

- `/start` — приветствие и главное меню
- **Запись** — пошаговый выбор: услуга → мастер → дата → время → подтверждение
- **Мои записи** — просмотр и отмена активных записей
- **Уведомление владельцу** — при каждой новой записи владелец получает сообщение в Telegram
- **SQLite** — данные хранятся локально, без сервера БД

```

└── README.md
```

## Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone git@github.com:404iq/beauty-salon-bot.git
cd beauty-salon-bot
```

### 2. Создать виртуальное окружение

```bash
python -m venv venv
# Windows:
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate
```

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

### 4. Настроить переменные окружения

Скопировать `.env.example` в `.env` и заполнить:

```
BOT_TOKEN=токен_от_BotFather
OWNER_CHAT_ID=ваш_chat_id  # узнать через @userinfobot
```

### 5. Запустить бота

```bash
python main.py
```

## Как узнать OWNER_CHAT_ID

Напишите боту [@userinfobot](https://t.me/userinfobot) — он пришлёт ваш числовой ID.
